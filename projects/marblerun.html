Marble Run

2026

Overview

The SWSC Science competition encompossing several events. My most notable win is from the Marble run event. In this event students are to construct a course where a marble travels along a track. The longest running track wins. Common stratagies are to create a series of switch-back tracks, maximizing total distance. Reducing the angle of the tracks to reduce acceleration also benefits run times. Common issues with these stratagies is the time-cost and risk of marbles getting stuck from friction. My solution is a mechanically governed system to transport the marble.

The Mechanism

The main mechanism of the marble run is a 3d printed lift system. Using a claw, the mechanism draws a pulley, raising the the marble to a higher landing platform. Once raised, the claw uses an additional mechanism to release the marble. The mechanism uses:

  • A driving weight -> provides constant force
  • A gear train -> amplifies the number of ticks per rotation
  • An escapment -> governs the angular velocity of the mechanism
